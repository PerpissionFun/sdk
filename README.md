## Installation

Clone repository:

```
git clone https://github.com/yourname/perpission.git
```

Enter folder:

```
cd perpission
```

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

---

## Environment Variables

Create `.env` file:

```
RPC_URL=your_rpc_endpoint
PRIVATE_KEY=your_private_key
```
